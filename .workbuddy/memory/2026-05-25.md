### 确认卡片 / formatValue 修复
- **体重数字字符串**：`formatValue` 中体重判断 `typeof value === 'number'` 扩展为同时接受数字字符串；`saveRecordToData` 中体重统一 `parseFloat` 处理
- **排便 times 字符串**：AI 返回 `"06:30、15:30"` 字符串而非数组，`formatValue` 和 `renderRecords` 都加了字符串分发处理
- **bowel 对象格式**：`formatValue` 中 bowel 为对象时返回 null，让子字段（times/count/note）各自显示
- **event 字段过滤**：确认卡片中 `event === '排便'` 等类别标记字段跳过显示
- **editRecord labelMap**：补充 `times`、`排便`、`bowel_movements`、`sleep_time`、`wake_time`、`起床时间`、`睡觉时间` 等中文映射
- **renderRecords 排便 times**：兼容 times 是字符串的情况，正确拆分显示
- **bowel_movements 字符串数组**：`formatValue` 中 `bowel_movements` 元素可能是字符串（如 `"06:30"`）而非对象，修复映射逻辑
- **renderRecords 单独 bowel_movements**：AI 有时只返回 `bowel_movements` 不返回 `bowel` 字段，renderRecords 加了独立处理路径
- **静态服务器语法错误**：`if (key === 'times' && typeof value === 'string' && value.includes(':');)` 多余分号导致整个 script 不执行，修复
